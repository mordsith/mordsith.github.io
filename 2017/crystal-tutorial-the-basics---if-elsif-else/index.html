Crystal Tutorial The Basics - If Elsif Else

  • February 1, 2017
  • 767 words
  • Sam Ashkan
  • crystal


The if, elsif and else are Crystals conditionals statements that execute a block of code to check if a condition is true of false. It works very similar to the english languge that if this is true i will do this else i will do something else.

Things to keep in mind!

= means you assing a value to the variable
== is for checking a conditional
&& means and
|| means OR

Problem: Lets say you want to output something if lets say the student is older than 19, and is to old to attend class.

code: 

student_name = 20

if student_name > 19
  puts "The student is to old for this course."
end
output: 
The student is to old for this course.

And we will get the desired output that the student is to old for this class. This is a very simple situation and we can do alot more with ifs, like using more of them.

Problem: Now lets say that the student can take the night course if he is older than 19 and younger than 25.

code: 

student_name = 20

if student_name > 19
   puts "The student is to old for this course."
  if student_name < 25
    puts "But the student can attend night courses"
  end
end
output: 
The student is to old for this course.
But the student can attend night courses

So how did we do this? We used something called a nested if. The first condition does the same as before. The second nested if gives the second option.

Problem: Output something if a patient has the flu, and if patient doesn't have it output something else. You only have the flu or you dont.

code: 

patient_has_flu = false

if patient_has_flu == true
  puts "The patient is ill"
else
  puts "The patiens is not sick!"
end
output: 
The patiens is not sick!

Notice we use == because we want to check if it is true, not give it a value. If all statements fail then the else condition is called.

Problem: To test alot of conditions we can use the else if conditions, in crystal we write elsif. So lets say that in our case we cant to figure out the persons age description.

code: 

person = 24

if person >= 0 && person <= 3
  puts "Toddler"
elsif person >= 4 && person <= 12
  puts "Child"
elsif person >= 13 && person <= 19
  puts "Teenager"
elsif person >= 20 && person <= 70
  puts "Adult"
elsif person >= 70 && person <= 110
  puts "Senior"
else
  puts "You are a turtle"
end
output: 
Adult

This way we can have many if and elsif statements. Keep in mind that the case control expression is better for pattern matching. But thats coming up in another tutorial.

Tip! If can also be written as an expression suffix.

code: 

x = 22
output = "Mathew" if x == 22
puts output

# And a ternary If looks like this...

puts a = 110 > 222 ? true : false
output: 
Mathew
false

Tip! You to check the variable type.

code: 

a = 22
if a.is_a?(String)
  puts "a is a string"
elsif a.is_a?(Number)
  puts "a is a number"
end
output: 
a is a number

And then there is the OR statement

code: 

a = 22
if a.is_a?(String) || a.is_a?(Number)
  puts "a is a number or a string"
end
output: 
a is a number or a string

And those where the basics of if

Success! You made it this far, i hope this helped to understand the basics of the conditional statements.

Summary

# This is what the If
# statement looks like.

if condition
  do_something
elsif
  do_something
else
  do_something
end

# A simple If statement
# if true do something

if condition
  do_something
end

# Nested If's
# if true do something
# and if second is true
# do that.

if condition
  do_something

  if condition   
    do_something
  end

end

# The Else statement
# if above conditions are
# not true do else.

if condition
  do_something
else              
  do something
end

# If more than one condition needs to be true
# if first and second condition is true
# do the code. else if the third and fourth
# condition is true do that. Otherwise
# do else.

if condition && condition   
  do_something
elsif condition || condition          
  do_something
else                                  
  do_something
end

# As expression suffix

x = 10 if condition                   

# Check if the variable type
# is a String.

if x.is_a?(String)                    
  do_something
end

# Check if the variable type
# is a Number.

if x.is_a?(Number)                    
  do_something
end
