Crystal Tutorial The Basics - Hash

  • February 8, 2017
  • 395 words
  • Sam Ashkan
  • crystal


In the last tutorial we discussed Arrays, in this tutorial im going to show Hashes. Hash is very similar to an array. The main differences are that an array is an integer indexed collections of object while hash points to a Key. This is what a Hash looks like…

code: 

hash = { key1: :value1 }

To create an empty Hash we must always define the K (key) And V (value).

code: 

# hash of Int32's
my_hash = {} of Int32 => Int32

# string and int based
my_hash = {} of String => Int32

# Same as above
my_hash = Hash(String, Int32).new

And to write in implicit form, easiest way to write and read imo

code: 

age = { "Martin Johnsson" => 29, "Jane Doe" => 16 }
pp age
output: 
age # => {"Martin Johnsson" => 29, "Jane Doe" => 16}

And an alternative syntax form.

code: 

options = { :font_size => 20, :font_family => "Arial" }
pp options
output: 
options # => {:font_size => 20, :font_family => "Arial"}

There’s alot of ways to write you’re hash, you should pick the way that suites you.

Problem: Output a specific key!

code: 

# output a specific key in the hash.
pp options[:font_size]
output: 
options[:font_size] # => 20

Problem: Create a list of movies and output movie name after the year it was made.

code: 

movie = {
  1972 => "The Godfather",
  1994 => "The Shawshank Redemption",
  1993 => "Schindler's List",
  1980 => "Raging Bull",
  1942 => "Casablanca"
}

pp movie[1980]
output: 
movie[1980] # => "Raging Bull"

Now keep in mind that this is not a good in real life example since there are alot more movies made the same year. But lets continue with it for learning.

Problem: Loop through the movie Hash list and output the result.

code: 

movie.each do |key, value|
  puts "Year: #{key} Movie: #{value}"
end
output: 
Year: 1972 Movie: The Godfather
Year: 1994 Movie: The Shawshank Redemption
Year: 1993 Movie: Schindler's List
Year: 1980 Movie: Raging Bull
Year: 1942 Movie: Casablanca

Summary of Hash

code: 

# Creating a hash
my_hash = { Key => Value, Key => Value }

# Clearing a hash.
my_hash = { 21 => "Hash" }
my_hash.clear

# Cloning a hash.
my_hash = { 21 => "Hash" }
my_sec_hash = my_hash.clone

# Check if its empty
my_hash.empty?

# Inverting the hash
{"foo" => "bar"}.invert
