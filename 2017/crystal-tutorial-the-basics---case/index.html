Crystal Tutorial The Basics - Case

  • February 3, 2017
  • 469 words
  • Sam Ashkan
  • crystal


So in the last tutorial we discussed if’s. This time im going to talk about the case statement. What is case and why should i care? Usually if you have more than a few conditionals case would be preferred. For the programmer case makes it alot easier to read and debug. You could think of it abit like pattern matching.

code: 

# what case looks like

case expression
when expression1, expression2
   statement
when expression3, expression4
   statement2
else
   statement3
end

This is what a case control expression looks like. It leads to better readability.

Problem: create a simple case, if the variable is met run something.

code: 

x = 1

case x
when 1
  puts "1"
when 2
  puts "2"
else
  puts "other"
end
output: 
2

This is a very simple case condition, when a condition is met the output is run. If it fails run else. And now lets say you have many options you want to do the same condition.

code: 

num = 22

case num
when 32, 11, 43, 22
  puts num
when "hello"
  puts "well hello there"
end

And then we can invoke methods in cases.

code: 

num = 11

case num
when .even?
  puts "even"
when .odd?
  puts "odd"
end

Problem: Lets say that you need to have a case condition between 1 and 100, instead of writing every single number we can do this.

code: 

case 5
when 1..100
  puts "The case statement in within range!"
end

And then we can check the variable type with case.

code: 

var = 10

case var
when String
  puts "String"
when Int32
  puts "32 bit integer"
else
  puts "Something else"
end

Problem: Now we want to fint the Capital of the Country we specify.

code: 

# Find the Capital of the country
country = "Germany"

capital = case country
  when "China" then "Beijing"
  when "India" then "New Delhi"
  when "Japan" then "Tokyo"
  when "Phillipines" then "Manila"
  when "Germany" then "Berlin"
  when "France" then "Paris"
  when "Sweden" then "Stockholm"
  when "Russia" then "Moscow"
  when "Egypt" then "Cairo"
  when "Spain" then "Madrid"
else "Unknown"
end

puts "The capital of " + country  + " is "  + capital
output: 
The capital of Germany is Berlin

And there is a little more of a real world use of case when you have alot of things to check. You could do it with if and elsif but it would not be recommended.

Summary

code: 

# what case looks like

case expression
when expression1, expression2
   statement
when expression3, expression4
   statement2
else
   statement3
end
code: 
# many checks

case num
when 32, 11, 43, 22
  do_something
when "string"
  do_something
end
code: 
# checking variable type

case var
when String
  do_something
when Int32
  do_something
else
  do_something_else
end
code: 
# When Then

var1 = case var2
  when something then do_something
else do_something_else
end
