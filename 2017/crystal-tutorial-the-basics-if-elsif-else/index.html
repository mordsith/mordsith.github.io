Crystal Tutorial The Basics If Elsif Else


  • February 1, 2017
  • 679 words
  • Sam Ashkan
  • crystal


The if, elsif and else are Crystals conditionals statements that execute a block of code to check if a condition is true of false. It works very similar to the english languge that if this is true i will do this else i will do something else.

Things to keep in mind!

  • = means you assing a value to the variable
  • == is for checking a conditional
  • && means and
  • || means OR

Example: lets say you want to output something if lets say the student is older than 19, and is to old to attend class.

Code: 

student_name = 20

if student_name > 19
  puts "The student is to old for this course."
end
Output: 
The student is to old for this course.

Output: We will get the output that the student is to old for this class.

Test: But now lets say that the student can take the night course if he is older than 19 and younger than 25. We can use nested if’s.

Code: 

student_name = 20

if student_name > 19
   puts "The student is to old for this course."
  if student_name < 25
    puts "But the student can attend night courses"
  end
end
Output: 
The student is to old for this course.
But the student can attend night courses

Output: The first condition does the same as before. The second nested if gives the second option.

Test: Now we want to output if a patient has the flu, and if patient doesn’t output something else. You only have the flu or you dont.

Code: 

patient_has_flu = false

if patient_has_flu == true
  puts "The patient is ill"
else
  puts "The patiens is not sick!"
end
Output: 
The patiens is not sick!

Output: If all statements fail then the else condition is called.

Test: To test alot of conditions we can use the else if conditions that in crystal we write elsif. So lets say that in our case we cant to figure out the persons age description.

Code: 

person = 24

if person >= 0 && person <= 3
  puts "Toddler"
elsif person >= 4 && person <= 12
  puts "Child"
elsif person >= 13 && person <= 19
  puts "Teenager"
elsif person >= 20 && person <= 70
  puts "Adult"
elsif person >= 70 && person <= 110
  puts "Senior"
else
  puts "You are a turtle"
end
Output: 
Adult

Output: This way we can have many if cases. Keep in mind that the Case control expression is better for pattern matching.

Example: If can also be written as an expression suffix.

Code: 

x = 22
output = "Mathew" if x == 22
puts output

# And a ternary If looks like this...

puts a = 110 > 222 ? true : false
Output: 
Mathew
false

Example: To check the variable type…

Code: 

a = 22
if a.is_a?(String)
  puts "a is a string"
elsif a.is_a?(Number)
  puts "a is a number"
end
Output: 
a is a number

Example: And then there is the OR statement

Code: 

a = 22
if a.is_a?(String) || a.is_a?(Number)
  puts "a is a number or a string"
end
Output: 
a is a number or a string

Summary

Code: 

# What If looks like..
if condition
  do_something
elsif
  do_something
else
  do_something
end

# A simple If statement
if condition                          # If this is true
  do_something                        # Do this
end

# Nested If's
if condition
  do_something

  if condition                        # If inside another if
    do_something
  end

end

# The Else statement
if condition
  do_something
else                                  # Else if above conditions fail
  do something
end

# If more than one condition needs to be true
if condition && condition             # If first AND second condition is true
  do_something
elsif condition || condition          # Else if first OR second condition is true
  do_something
else                                  # If above code fails
  do_something
end

# As expression suffix
x = 10 if condition                   # Variable is 10 if condition met

# Check the variable type
if x.is_a?(String)                    # If variable x is a String
  do_something
end

if x.is_a?(Number)                    # If variable x is a Number
  do_something
end
