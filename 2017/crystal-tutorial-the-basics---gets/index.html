Crystal Tutorial The Basics - Gets


  • January 31, 2017
  • 522 words
  • Sam Ashkan
  • crystal


In the last tutorial we output text and numbers, in this tutorial im going to talk about something called standard streams. There are three kinds of standard streams and there are called …

  • stdin - standard input
  • stdout - standard output
  • stderr - standard error

Last time we used stdout, these where puts and print to output on screen in Crystal. This time im going to discuss stdin, meaning getting input from the user. Usually in Crystal lang this is done by the command gets. This is what a simple gets looks like…

Code: 

x = gets          # stores the input to x
puts x            # outputs x
Output: 
Crystal 123       # my input
Crystal 123       # what crystal outputs

Now this is a very simple way to do it compared to some other languages.. * coughs rust * . I know that you usually don’t use something like input in real world software but it is very important when learning in my opinion.

To get a sertain amount of characters you can do this…

Code: 

puts gets(3)      # outputs the three first characters
Output: 
Samuel L Jackson  # what i put in
Sam               # what gets took in

To stop taking in to the stdin when a specific character is put in you can ..

Code: 

puts gets('s')      # will stop after the char s in found
Output: 
today i had a sandwich for lunch # my input
today i had a s

You can also stop after a sertain amount of chars if you write puts gets(’s’, 10). To read before the delimiter is found ..

Code: 

puts gets("c++", 20)      # will stop before the delimiter is found
Output: 
i do alot of c++ code
i do alot of

Its easy to split up the gets. There is alot of information about this topic and more on Crystal Lang Api site. According to the site a not_nil! command is required that the compiler forces us to deal with. Like this …

Code: 

puts gets.not_nil!.split
Output: 
The Earth is Round
["The", "Earth", "is", "Round"]

And to split up to two variables.

Code: 

x, y = gets.not_nil!.split
puts x + " " + y
Output: 
11 22 33         # input
11 22            # output, notice it did not take in the last numbers

We can also split with a perticular character.

Code: 

puts gets.not_nil!.split('-')           # split when -
Output: 
Linda-Samantha-Nick-Bjorn               # my input
["Linda", "Samantha", "Nick", "Bjorn"]

And lastly if we want to add numbers we can do this one liner…

Code: 

puts gets.not_nil!.split.map(&.to_i).sum
Output: 
120 21 43 66 22         # input
272

And there is probably alot more you can do with gets. Got anything to add please let me know.

Summary

Code: 

x = gets                               # stores the input to x

gets(3)                                # stores the three first characters

gets('s')                              # will stop after the char s in found

gets("c++", 20)                        # will stop before the delimiter is found

gets.not_nil!.split                    # split up the input

x, y = gets.not_nil!.split             # split up input to two variables

gets.not_nil!.split('-')               # split when -

gets.not_nil!.split.map(&.to_i).sum    # split up to integers and sums
