Crystal Tutorial The Basics - Gets

  • January 31, 2017
  • 533 words
  • Sam Ashkan
  • crystal


In the last tutorial we output text and numbers, in this tutorial im going to talk about something called standard streams. There are three kinds of standard streams and there are called …

  • stdin - standard input
  • stdout - standard output
  • stderr - standard error

Last time we used stdout, these where puts and print to output on screen in Crystal. This time im going to discuss stdin, meaning getting input from the user. Usually in Crystal lang this is done by the command gets. This is what a simple gets looks like…

Code: 

# stores the input to x
# and then outputs x.

x = gets          
puts x
Output: 
Crystal 123
Crystal 123

Now this is a very simple way to do it compared to some other languages.. * coughs Rust * . I know that you usually don’t use something like input in real world software but it is very important when learning in my opinion. And for completing tasks like on Rosetta Code or Hackerrank.

Problem: Get the first 3 characters in gets.

Code: 

# gets input, but only the
# first three characters.

puts gets(3)
Output: 
Samuel L Jackson
Sam

To stop taking in to the stdin when a specific character is put in you can ..

Problem: Get input until you find the char s and include it in the output.

Code: 

# will stop after the
# char s in found

puts gets('s')
Output: 
today i had a sandwich for lunch
today i had a s

And like expected it stopped AFTER ’s’.

Tip! You can also stop after a sertain amount of chars if you write `puts gets('s', 10)`. To read before the delimiter is found.
Problem: Stop BEFORE the last character is found.

Code: 

# will stop before the
# delimiter is found

puts gets("c++", 20)
Output: 
i do alot of c++ code
i do alot of

Its easy to split up the gets. There is alot of information about this topic and more on Crystal Lang Api site. According to the site a not_nil! command is required that the compiler forces us to deal with. Like this …

Code: 

puts gets.not_nil!.split
Output: 
The Earth is Round
["The", "Earth", "is", "Round"]

And to split up to two variables.

Code: 

x, y = gets.not_nil!.split
puts x + " " + y
Output: 
11 22 33         
11 22

We can also split with a perticular character.

Code: 

# split when -

puts gets.not_nil!.split('-')
Output: 
Linda-Samantha-Nick-Bjorn              
["Linda", "Samantha", "Nick", "Bjorn"]

Tip! To sum numbers with a one liner.

Code: 

puts gets.not_nil!.split.map(&.to_i).sum
Output: 
120 21 43 66 22         
272

And there is probably alot more you can do with gets. Got anything to add please let me know.

Summary

Code: 

# storing the input to x

x = gets                               

# storing the first three characters

gets(3)                                

# will stop after the char s in found

gets('s')                              

# Stop before the delimiter is found

gets("input", 20)                        

# split up the input

gets.not_nil!.split                    

# split up the input

gets.not_nil!.split                    

# split up input to two variables

x, y = gets.not_nil!.split             

# split when - is found in the text

gets.not_nil!.split('-')               

# split up to integers and sums

gets.not_nil!.split.map(&.to_i).sum
