Crystal Tutorial The Basics - Puts

  • January 29, 2017
  • 887 words
  • Sam Ashkan
  • Crystal


I really like these new programming languages like Go and Rust. I found a relatively new language called Crystal. I am really amazed by its performance and how easy it is to use. Crystal has similar syntax as Ruby but compared to Ruby it is a compiled language and not interpreted, making it perform really well and leave a low memory footprint.

Note:

  • Crystal is Alpha stage at the moment.
  • Crystal does not work on Windows, but it will sometime in 2017.

To install Crystal you should take a look at the official documents called The Crystal Book! It is also a great way to learn more about the language.

To create a Crystal project type …

Terminal: 

$ crystal init app casual

crystal is the name of the compiler. The init command creates the default project structure. The app commands creates an app skeleton, to create a library change it to lib. casual is just the name i picked, you may change that to whatever you like. To run the app type …

Terminal: 

$ cd casual
$ crystal src/casual.cr

And nothing will happen since we haven’t put anything in the file.

Open the casual.cr file with your favorite editor located inside the src folder. Remove everything inside and type …

Code: 

puts "Hello, World!"

Save the file and return to the terminal and run it!

Output: 

$ crystal src/casual.cr
Hello, World!

And congrats! You’r first Crystal application. Totally worthless but still pretty cool.

To build the app type …

Terminal: 

$ crystal build src/casual.cr --release
$ ./casual
Hello, World!

Lets have a look at the file we just created.

Terminal: 

$ ls -shR casual
104K casual

104 kilobytes! That is really amazing for such a new complex language.. specially if you compare it to other new languages. I got it down to 80kb’s with strip. Thats like C small! I am not sure how Crystal manages this but i will try to find out.

Back to Programming

Here are some simple output commands. It is good to remember that…

  • puts adds a new line after the end.
  • print does not add a new line.

btw hashtag means that it is a comment.

Code: 

puts "Hello,"                     # simple output with new line
puts "World!"                     

print "Hello, "                   # without new line
print "World!\n"                  # \n = newline like in most C langs

my_name = "Sam"                   # declaring a string variable
puts "My name is: #{my_name}"     # output with a placeholder for the variable

Output: 

$ crystal src/casual.cr
Hello,
World!
Hello, World!
My name is: Sam

In my variable ( variables are placeholders ) i assigned a String with the value of Sam. Crystal being similar to Ruby figures out that it is a string. And then inside the hashtag curley brackets i can output the variable.

Lets do some other fun stuff with strings..

Code: 

# The String
my_text = "I Love Crystal"

# Nr of chars
puts "my_text has #{my_text.size} characters"

# Uppercase
puts "#{my_text.upcase}"                      

# Lowercase
puts "#{my_text.downcase}"                    

# substitutes o's with i's
puts "#{my_text.sub("o","i")}"                

# True! It does.
puts "#{my_text.ends_with?("Crystal")}"

Output: 

$ crystal src/casual.cr
my_text has 14 characters
I LOVE CRYSTAL
i love crystal
I Live Crystal
true

Something cool i found was the “Heredoc”. I had never heard of it before but seems really useful for writing large string, or if you have alot of symbols. Heredoc starts with <<- and the name you give it. Then the text has to be indented 4 line.

Code: 

# The text has to be indented 4 lines

x = <<-SOMETHING
    Hello to                    
    /"You People In
    In The " " World?=)()
    "{''¤%&¤{&/(%/&)}}"
    SOMETHING

puts x

Output: 

$ crystal src/casual.cr
Hello to                          
/"You People In
In The " " World?=)()
"{''¤%&¤{&/(%/&)}}"

And of course there is alot more you can do…

Mathematics

Now for some simple math. Doing simple mathematics with crystal is really simple i think.

Code: 

# add
puts 1 + 3                

# substract
puts 10 - 3               

# divide
puts 20.0 / 2.0           

# multiply
puts 71 * 13              

# modulus "not percent"
puts 231 % 10             

# less than
puts 22 < 10              

# more than
puts 22 > 10              

# more or equal
puts 33 >= 15             

# and we can add inside the string
puts "11.1 / 22.4 = #{11.1/22.4}"

Output: 

$ crystal src/casual.cr
4
7
10.0
923
1
false
true
true
11.1 / 22.4 = 0.4955357142857143

Problem: And now let's convert Celcius to Fahrenheit. 0 degrees Celcius is 32 degrees Fahrenheit. To convert temperatures in degrees Celsius to Fahrenheit, multiply by 1.8 (or 9/5) and add 32.

Code: 

celsius = 32;
fahrenheit = 1.8 * celsius + 32

puts "32 degrees celsius is #{fahrenheit}"

Output: 

$ crystal src/casual.cr
32 degrees celsius is 89.6 degrees fahrenheit

And of course alot more complicated mathematics can be performed, and i’m not a mathematician, but you get the picture.

We have created a Crystal project, Compiled it, outputted code and done some simple mathematics. I hope this will help you to learn a bit about The Crystal language and Puts. Crystal syntax as i said is based on Ruby where there’s tons of information. I am new to this language aswell and have never tried Ruby before, but it seems like a lang that makes developing a bit more simple.
